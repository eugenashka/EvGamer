# EvGamer

Данная игра направлена на  изучение Python. Проект выкладывается с целью демонстрации на собеседованиях.
В данный момент не подготовлены условия задач, поэтому ответ на большинство задач - их номер. 

Планируется расширение с добавлением задач на понимание многопоточного программирования, работы с библиотеками NumPy, Pandas, работы с файлами JSON, XML и т.п., работы с сетевыми подключениями, API.
